## 为什么选了 Astro

之前看了 [ninehills.tech](https://ninehills.tech) 这个博客，研究了一圈方案：

- **Jekyll** — Ruby 环境在国内装起来烦，构建慢
- **Hexo** — 生态老化，很多主题不维护了
- **Hugo** — 很好但模板语法怪
- **WordPress** — 太重，要服务器要数据库

最终选了 **Astro**：写 Markdown → push 到 GitHub → 自动部署。零维护。
