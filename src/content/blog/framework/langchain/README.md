# LangChain (2025 Edition)

## 🌟 2025 年的 10 月 LangChain 正式发布了其V1.0的版本, 对其内容和架构做出了巨大的更新

截止 2025 年，LangChain 已从最初的大杂烩式工具集，进化为一个专注于 **构建可适应生态进化的 Agent 平台**。
其核心理念已从“链（Chain）”全面升级为“智能体架构（Agent Architecture）”。
